# CLI-инструмент для дизеринга изображений (в разработке)

Поддержка форматов: .jpeg, .png, .gif

## Флаги и пример использования
* -h / --help : Вывод справки
* -s / --source : Путь к изображению или директории с изображениями для обработки. Значение по умолчанию: текущая директория.
* -d / --destination : Путь к директории для обработанных изображений. Значение по умолчанию: текущая директория.
* --suffix : Суффикс имени файла обработанного изображения. Значение по умолчанию: "-dither". 
* -f / --function : Алгоритм обработки изображения. Значение по умолчанию: "threshold".
* --threshold : Значение порога для алгоритма порогового дизеринга. Допутимое значение: от 0 до 255. Значение по умолчанию: 128.
* --matrix : Размер матрицы Байера. Допустимые размеры: 2, 4, 8, 16. Значение по умолчанию: 2.

Пример использования:
app --source="./images/image.png" --destination="./images/dither/" --suffix="-new"
Обработанное изображение будет находится по адресу ./images/dither/image-new.png
